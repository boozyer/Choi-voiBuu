# Design & 3D Render Workflow Scripts

Kho lưu trữ các script lặt vặt hỗ trợ tự động hóa trong quá trình làm thiết kế và xử lý file 3D. 

## Công cụ hiện có
- **batch_rename_renders.py**: Script Python giúp tự động đổi tên và đánh số thứ tự hàng loạt cho chuỗi ảnh sequence khi xuất ra từ Octane Render hoặc After Effects. Hỗ trợ dọn dẹp file trước khi đưa vào hậu kỳ.
